# Shell I/O Redirections and Filters

This directory contains scripts for the ALX Shell I/O Redirections and Filters project.

## Tasks

0. Hello World - Print "Hello, World"
1. Confused Smiley - Display a confused smiley
2. Let's Display a File - Display /etc/passwd
3. What About 2? - Display /etc/passwd and /etc/hosts
4. Last Lines of a File - Display last 10 lines of /etc/passwd
5. I'd Prefer the First Ones, Actually - Display first 10 lines of /etc/passwd
6. Line #2 - Display the third line of iacta
7. It's a Good File That Cuts Iron Without Making a Noise - Create a file with special characters
8. Save Current State of Directory - Write ls -la output to file
9. Duplicate the Last Line - Duplicate last line of iacta
10. No More Javascript - Delete all .js files
