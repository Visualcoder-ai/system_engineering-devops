# Shell Permissions

This project covers basic file permissions management in Linux/Unix systems.

## Tasks

- Task 0: Switch user to betty
- Task 1: Print effective username
- Task 2: Print user groups
- Task 3: Change file owner
- Task 4: Create empty file
- Task 5: Add execute permission
- Task 6: Set specific permissions (James Bond)
- Task 7: Set complex permissions (John Doe)
- Task 8: Add execute to directories
- Task 9: Create directory with specific permissions
- Task 10: Change group owner
