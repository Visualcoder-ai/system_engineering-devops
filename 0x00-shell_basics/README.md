# 0x00-shell_basics

This directory contains basic shell scripting exercises for the ALX Foundations of Linux and Version Control program.

## Tasks

### 0. Where Am I?
- **File:** 0-current_working_directory
- **Description:** Write a script that prints the absolute path name of the current working directory.
- **Command:** pwd

### 1. What's in There?
- **File:** 1-listit
- **Description:** Write a script that displays the contents list of your current directory.
- **Command:** ls

### 2. There's no Place Like Home
- **File:** 2-bring_me_home
- **Description:** Write a script that changes the working directory to the user's home directory.
- **Command:** cd ~

### 3. The Long Format
- **File:** 3-listfiles
- **Description:** Write a script that displays the current directory contents in a long format.
- **Command:** ls -l

### 4. Hidden Files
- **File:** 4-listmorefiles
- **Description:** Write a script that displays the current directory contents, including hidden files, in long format.
- **Command:** ls -la

## Requirements

- All scripts must start with the shebang: `#!/bin/bash`
- All scripts must be executable
- Each script must perform exactly what is required
